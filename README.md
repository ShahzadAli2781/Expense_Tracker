# Expense Tracker (Desktop-Based Python Application)

A simple and beginner-friendly desktop-based Expense Tracker developed using Python.
This console application allows users to enter daily expenses, calculate the total amount spent, and view an expense summary with average spending.

## 📌 Features

* Add multiple expense entries
* Calculate total amount spent
* Calculate average expense
* Input validation
* Prevent negative values
* Beginner-friendly Python logic
* Clean console-based user interface

## 🛠️ Technologies Used

* Python 3

## 📚 Concepts Used

* Variables
* Loops
* Conditional Statements
* Exception Handling
* Mathematical Operations
* Accumulator Logic

## 💻 Project Type

Desktop-Based Console Application
